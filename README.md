# IBM HR Analytics - Employee Attrition Dashboard

## Project Overview
This project delivers an end-to-end analysis of employee attrition data using SQL and Power BI. The primary objective is to identify key organizational turnover drivers—such as job roles, departments, income levels, and gender dynamics—enabling human resources to make data-driven retention decisions.

## Key Dashboard Metrics
- Total Employee Count: 1K (1,470 Employees)
- Total Attrition: 237
- Attrition Rate: 16.12%

## Repository Content
- HR_Analytics_Dashboard.pbix: Main Power BI file containing the data model, DAX measures, and interactive visualizations.
- HR_Analytics_Dashboard.pdf: A high-quality PDF export of the dashboard report for quick, software-free viewing.
- HR_Analytics_Project_Quries.sql: SQL scripts utilized for data cleaning, validation, and database-level metrics verification.
- WA_Fn-UseC_-HR-Employee-Attrition.csv: The source IBM Watson raw dataset.

## Tech Stack and Skills Demonstrated
- Database Management (SQL): Data aggregation, filtering, null-handling, and business logic implementation.
- Data Visualization (Power BI): Developing interactive charts, implementing dynamic slicers (Gender/JobLevel), establishing a professional UI theme, and structuring KPI cards.
- HR Analytics & Insights: Translating raw workforce data into actionable corporate talent insights.
